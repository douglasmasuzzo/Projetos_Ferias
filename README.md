# **Repositório Projeto_Ferias** # 
Estudos Práticos em Desenvolvimento de Software (Front-end, Back-end e Client–Server)

------------------------------------------------------------------------------------

## **Sobre o Repositório** ##
O repositório Projeto_Ferias foi criado com a finalidade de centralizar experimentos, estudos práticos e projetos acadêmicos desenvolvidos ao longo do período de recesso, com foco na consolidação de conceitos fundamentais de desenvolvimento de software.
Inicialmente, o repositório era voltado exclusivamente a experimentos em Front-end, porém, ao longo de sua evolução, passou a incorporar projetos com arquitetura Client–Server, envolvendo Back-end, consumo de APIs REST e integração entre diferentes tecnologias.

### _Evolução do Escopo_ ###

- _Fase Inicial – **Front-end**_
  - HTML
  - CSS
  - JavaScript

( ~~Ênfase em estruturação de páginas, estilização e interatividade básica~~ ) 

- _Fase Atual – **Client–Server**_
  - APIs REST com Node.js
  - Banco de dados
  - Consumo de API por aplicações cliente (Java)
  - Comunicação via HTTP e JSON

( Ênfase em separação de responsabilidades, arquitetura em camadas e integração entre sistemas )

-------------------------------------------------------------------------------------------------

# **Conclusão** #

O Projeto_Ferias representa um espaço de aprendizado contínuo, onde conceitos teóricos são testados na prática, permitindo a compreensão real de como aplicações modernas são estruturadas, comunicam-se entre si e evoluem ao longo do tempo. O repositório permanece aberto a expansões futuras, incluindo novas linguagens, frameworks e abordagens arquiteturais.
